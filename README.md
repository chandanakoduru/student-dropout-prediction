# 🎓 Student Dropout Prediction System

## 📌 Project Overview
This project predicts student dropout risk using Machine Learning and Deep Learning techniques based on educational behaviour and engagement patterns.

The system analyzes:
- Student academic performance
- Learning platform activity
- Assessment behaviour
- Temporal engagement patterns

to identify students who are at risk of dropping out.

---

## 🚀 Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- CatBoost
- TensorFlow / Keras
- Graph Neural Networks (Exploratory)

---

## 📂 Dataset
This project uses the **OULAD (Open University Learning Analytics Dataset)**.

Dataset includes:
- Student demographic data
- Assessment records
- Virtual learning environment interactions
- Course information
